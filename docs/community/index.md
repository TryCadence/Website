# Community

Join the Cadence community! Whether you're reporting bugs, suggesting features, or contributing code, we welcome your participation.

## Quick Navigation

### [Contributing Guide](/docs/community/contributing)
Complete guide to contributing to Cadence. Includes code style guidelines, PR process, and development setup.

### [Security Policy](/docs/community/security)
Security best practices and responsible disclosure policy. Report vulnerabilities and learn about Cadence's security features.

## Get Involved

### Report Issues
Found a bug? [Open an issue](https://github.com/TryCadence/Cadence/issues) on GitHub with reproduction steps.

### Request Features
Have an idea? [Start a discussion](https://github.com/TryCadence/Cadence/discussions) or create a feature request issue.

### Contribute Code
Ready to code? Check out our [Contributing Guide](/docs/community/contributing) and find [good first issues](https://github.com/TryCadence/Cadence/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22).
